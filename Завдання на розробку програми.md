# Завдання на створення програми:
Створити програму, яка буде читати дані із файлу БД MySQL
та дозволяти користувачеві вводити нові записи, редагувати існуючі
записи та робити обробку даних за заданими параметрами.
Результати відбору потрібно зберігати у файлі MS Word.
Файл БД MySQL містить довідки про житло. Кожен запис має
поля: прізвище, адресу, загальну площу житла. Кількість записів
довільна (не менше 10).
Програма повинна видавати у файл MS Word інформацію
такого вигляду:
▪ за прізвищем, що вводиться, видаються адреса та площа
житла;
▪ визначається кількість сімей, що мають житлову площу,
більшу ніж ХХ (вводиться у форму).
